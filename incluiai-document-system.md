# 📘 IncluiAI — Design System Documental (Oficial)

## 🎯 Objetivo

Padronizar todos os documentos gerados pelo IncluiAI com um visual:

* profissional
* moderno
* soft premium
* institucional (sem ser burocrático)
* pronto para PDF e uso real (escolas, INSS, relatórios oficiais)

---

# 🧠 Princípios do Sistema

1. **Clareza acima de tudo**
2. **Leitura fácil para pais e instituições**
3. **Visual leve (não poluído)**
4. **Hierarquia bem definida**
5. **Consistência entre todos os documentos**

# ⚠️ Regras Importantes

* NÃO usar texto solto → sempre usar cards
* NÃO poluir visual
* NÃO repetir informação
* NÃO gerar páginas em branco
* NÃO usar linguagem robótica

---

# 🚫 Alunos Incompletos (Regra de Sistema)

Aluno incompleto deve:

* ter borda vermelha
* fundo leve vermelho
* mostrar: "⚠ Completar cadastro"
* ter botão: "Completar cadastro"

E NÃO pode:

* gerar documentos
* usar IA
